1. What is a Stochastic Process?

A stochastic process is a collection of random variables indexed by time or space. Formally, a stochastic process is written as:

$$ \{X_t : t \in T\} $$

where \( T \) is the index set (often representing time), and \( X_t \) is a random variable representing the state at time \( t \).

2. Key Concepts

  • State Space: Set of all possible values the process can take.
  • Index Set: Usually time – can be discrete or continuous.
  • Sample Path: A realization of the stochastic process.

3. Types of Stochastic Processes

  • Discrete-Time vs Continuous-Time
  • Discrete-State vs Continuous-State
  • Markov Processes
  • Poisson Processes
  • Brownian Motion

4. Applications

Stochastic processes are used in:

  • Financial modeling (e.g., stock prices)
  • Queueing theory
  • Population dynamics
  • Reliability and survival analysis
